# Установка Linux

Для установки ubuntu используйте образ с [официального сайта](https://ubuntu.com/download/desktop).

На windows рекомендуется использовать виртуальную машину [virtual box](https://www.virtualbox.org/).
На macos рекомендуется использовать [UTM](https://mac.getutm.app/).

> [!warn]
> Если архитектура вашего процессора не `x86` (в частности если у вас
> macbook с процессором серии M*), у вас могут возникнуть трудности с решением задач
> на assembler. Для их решения вам придется либо устанавливать и запускать ubuntu с
> эмуляцией архитектуры (что сильно ухудшит производительность), либо найти машину
> с `x86` процессором.
