# Parallel implication

В этой задаче нужно вычислить следующую импликацию, распределив работу между потоками.

$(((a_1 \to a_2) \to a_3) \to ... )$

Для распределения работы используйте `std::thread` - подумайте, каким образом оптимальнее всего
распределить работу. Рассчеты делайте с учетом того, что в тестирующей системе вам будет доступно всего 4 ядра.

Напишите решение в файле [implication.cpp](./implication.cpp).
