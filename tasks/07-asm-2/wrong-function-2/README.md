# Wrong function 2

Студент Миша ошибся при написании своей функции `uint64_t process(...)` и написал пролог* дважды. При этом без этой ошибки его функция корректна и возвращает некоторое число. Но вот незадача - переписать его исходный код вы не можете. Аргументы функции неизвестны, так как вам было лениво вчитываться в Мишин код, и могут быть призвольными.
Напишите функцию `call_process` в файле [solution.S](./solution.S), которая вызовет функцию `process` ровно один раз и вернет возвращаемое число.

*Пролог который продублировался:
```asm
push ebp
mov ebp, esp
```

---

Например, если Мишина функция такова:
```
push ebp
mov ebp, esp
push ebp
mov ebp, esp
mov eax, 4
mov esp, ebp
pop ebp
ret
```
то ваша фунция должна вернуть число 4.
