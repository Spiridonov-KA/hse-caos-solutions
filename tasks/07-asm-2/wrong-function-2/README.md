# Wrong function 2

Студент Миша ошибся при написании своей функции `uint64_t Process(...)` и написал пролог* дважды.
При этом без этой ошибки его функция корректна и возвращает некоторое число. Но вот незадача –
переписать его исходный код вы не можете. Аргументы функции неизвестны, так как вам было лениво
вчитываться в Мишин код, и могут быть призвольными.

Напишите функцию `Solve` в файле [solution.S](./solution.S), которая вызовет функцию `Process`
ровно один раз и вернет возвращаемое число.

*Пролог который продублировался:

```asm
push rbp
mov rbp, rsp
```

---

Например, если Мишина функция такова:

```asm
push rbp
mov rbp, rsp
push rbp
mov rbp, rsp
mov rax, 4
mov rsp, rbp
pop rbp
ret
```

то ваша фунция должна вернуть число 4.
