# Traverse

Вашей программе в качестве первого аргумента командной строки передается путь до директории `P`.
Выведите все содержимое этой директории, которое можно пронаблюдать с помощью `ls`, в
произвольном порядке. Не выводите скрытые файлы. Скрытые директории и символические ссылки
выводить нужно. Содержимое директории следует выводить в виде

```plain
t1 path1
t2 path2
...
```

Где `ti` – тип данного вхождения – `d/f/l` для директорий, файлов и символьных ссылок, а `path` –
путь до данного файла/директории/символической ссылки **относительно директории `P`**.

Например, если директория `P` имеет следующую структуру:

```
$ tree P
P
|-- a
|-- b
|   `-- c
`-- d
    `-- e
        `-- f
            `-- g
```

Одним из возможных выводов будет

```
f a
d b/
f b/c
d d/
d d/e/
d d/e/f/
f d/e/f/g
```

> [!note]
>
> Используйте [`readdir`](https://www.man7.org/linux/man-pages/man3/readdir.3.html) для обхода директорий.

О том, как работать с аргументами командной строки можно почитать в условии задачи [broken-symlinks](../broken-symlinks/README.md).
