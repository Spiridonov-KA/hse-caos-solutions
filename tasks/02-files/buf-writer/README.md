# Буферизованный writer

Реализуйте буферизованный writer.

Writer создается от двух аргументов – файловый дескриптор, над которым делается буферизованный вывод, и размер буфера. В процессе свой работы writer не должен совершать динамических аллокаций – они разрешены только при создании writer'а. Writer должен иметь два метода:

- `int Write(std::string_view data)` – дописывает `data` в конец текущего буфера, при необходимости совершает `Flush`
- `int Flush()` – принудительно сбрасывает содержимое текущего буффера в переданный файловый дескриптор

Если во время работы какого-либо из методов произошла ошибка записи, метод должен вернуть `-errno`, в противном случае метод должен вернуть `0`.

> [!note]
> Writer не должен владеть переданным файловым дескриптором.
> То есть после вызова деструктора writer'а, файловый дескриптор должен продолжить существать.
