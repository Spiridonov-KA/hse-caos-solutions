# Захват вывода

Реализуйте функцию

```cpp
template <class F>
std::variant<std::pair<std::string, std::string>, int>
CaptureOutput(F&& f, std::string_view input);
```

Которая вызывает функцию f и захватывает ее поток вывода и поток ошибок, подавая ей на стандартный поток ввода данные из `input`.

В случае успеха верните пару из содержимого стандартного вывода и стандартного потока ошибок
(первая опция в `std::variant`). В случае возникновения ошибки, верните ее код (вторая опция в `std::variant`).

Гарантируется, что размер `input`, а так же размер стандартного вывода и потока ошибок функции `f` не превосходит `64KiB`. Гарантируется, что `f` не бросает исключения.

После завершения вызова `CaptureOutput`, состояние стандартных потоков ввода-вывода не должно измениться.
Так же, не допускайте утечки файловых дескрипторов. В этой задаче запрещено создание новых процессов и потоков.
Постарайтесь так же обойтись без создания новых файлов (в том числе в процессе работы `CaptureOutput`).
