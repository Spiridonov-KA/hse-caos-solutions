# Char sum

Напишите программу, которая считывает со стандартного потока ввода последовательность байт, выделяет те байты, которые являются кодами десятичных цифр в ASCII и выводит их сумму на экран в виде десятичного числа.

Не забывайте добавлять `\n` в конце вывода.

> [!note]
> Используйте системные вызовы [`read`](https://www.man7.org/linux/man-pages/man2/read.2.html) и [`write`](https://www.man7.org/linux/man-pages/man2/write.2.html) для работы со стандартными потоками ввода/вывода.

## Пример

Input:

```
1abcd23 4
```

Output:
```
10
```
