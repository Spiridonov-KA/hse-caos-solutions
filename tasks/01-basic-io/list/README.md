# Двувсязный список

Реализуйте двусвязный список – структуру данных для хранения последовательности элементов, в
которой каждый элемент хранится в отдельном "узле", который так же хранит указатель на следующий и предыдущий узел:

```plain
                        ┌──────────────┐         ┌───
                        │              ▼         │
      ┌────┬─────┬────┐ │      ┌────┬─────┬────┐ │
      │    │     │    │ │      │    │     │    │ │
... ┌─┤prev│value│next├─┘    ┌─│prev│value│next┼─┘ ...
    │ │    │     │    │      │ │    │     │    │
    │ └────┴─────┴────┘      │ └────┴─────┴────┘
    │         ▲              │
 ───┘         └──────────────┘
```

Эта структура, среди прочего, позволяет делать вставку и удаление
из любого места за O(1). От вас потребуется реализовать методы:

- `PushBack`
- `PushFront`
- `PopBack`
- `PopFront`
- `Back` – возвращает ссылку на последний элемент списка. Гарантируется, что список не пуст.
- `Front` – возвращает ссылку на первый элемент списка. Гарантируется, что список не пуст.
- `IsEmpty`
- `Swap` – меняет местами содержимое текущего списка и переданного.
- `Clear` – удаляет все элементы из списка и освобождает память.
- `Splice` – добавляет все содержимое переданного списка в конец текущего. Переданный список после этого должен стать пустым.
- `ForEachElement` – вызывает переданный функтор от всех элементов списка.

Все методы кроме `ForEachElement` и `Clear` должны работать за O(1), все методы кроме `PushFront` и `PushBack` не должны делать динамических аллокаций<sup>1</sup>. Так же, создание пустого списка не должно делать динамических аллокаций. Референсную реализацию на основе `std::list` вы можете найти в файле с тестами.

<sup>1</sup>динамические аллокации – использование ключевого слова new или функций malloc, calloc, realloc. Подробнее об этом вы узнаете дальше в курсе.
