# Mul8

Реализуйте функцию
```cpp
void Mul8(float *f);
```

В зависимости от значения `*f`, функция должна:

- Оставвить значение неизменным, если оно – бесконечность любого знака или NaN
- Записать в `*f` значение `8 * (*f)`, если оно представимо в float
- Записать в `*f` максимальное по модулю конечное представимое число того же знака,
  что и f, в остальных случаях.

Например, `Mul8(1.0f) == 8.0f`, а `Mul8(INFINITY) == INFINITY`.

Не используйте операции с вещественными числами, кроме операции присваивания
и взятия адреса. Не используйте ассемблерные вставки.
Стандартные функции для работы с вещественными числами также запрещены.

Решение будет тестироваться с флагами компилятора
`-mno-80387 -mno-sse -mno-sse2 -mno-avx -mno-fma -msoft-float`,
так что программы, использующие операции с вещественными числами, не скомпилируются.
