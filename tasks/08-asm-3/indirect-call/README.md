# Indirect call

Реализуйте функцию `Indirect.Call` в файле [`solution.S`](./solution.S), которая
принимает в регистрах `rax`, `rbx` и `rcx` указатели на функции с сигнатурами
`uint64_t(uint64_t, uint64_t)`, `uint64_t()` и `uint64_t()` соответственно.

`Indirect.Call` должна вызывать функции, которые переданы в `rbx` и `rcx`
(в произвольном порядке), а далее передавать результаты их работы в
функцию, переданную в `rax` (результат работы функции, переданной в
`rbx` должен идти в первый аргумент). Все 3 функции удовлетворяют соглашению
о вызовах System V ABI. `Indirect.Call` должен сохранять значение
**всех регистров общего назначения** за исключением `rax`, в котором но возвращает результат.
