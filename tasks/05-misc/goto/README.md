# Goto

Скорее всего, вы знакомы с оператором `goto`. Он позволяeт перейти в программе на заданную метку. `goto`, в числе прочего, нередко используется в C для освобождения ресурсов при обработке ошибок:

```c
err = ...

if (err) {
    goto free_resources;
}

// logic if no error

free_resources:
    // ...
```

(В языке C++ для избегании утечки ресурсов следует пользоваться [идиомой RAII](https://en.cppreference.com/w/cpp/language/raii.html))

В этой задаче мы потренируемся работать с этим оператором, реализовав [конечный автомат](https://ru.wikipedia.org/wiki/%D0%9A%D0%BE%D0%BD%D0%B5%D1%87%D0%BD%D1%8B%D0%B9_%D0%B0%D0%B2%D1%82%D0%BE%D0%BC%D0%B0%D1%82).

В качестве задания изобразите автомат, который будет считать количество подстрок "os" и "rop" в строке.

Далее, используя этот автомат, имплементируйте функцию `CountSubstrings` в файле [count-substrings.cpp](./count-substrings.cpp): с помощью меток закодируйте ваши состояния, с помощью оператора `goto` реализуйте переходы. Функции на вход будет передаваться строка, состоящая из строчных латинских букв.

Также в комментариях в файле решения предоставьте ссылку на рисунок автомата, который вы имплементировали.

В файле [example.cpp](./example.cpp) находится пример автомата для подсчета количества подстрок
