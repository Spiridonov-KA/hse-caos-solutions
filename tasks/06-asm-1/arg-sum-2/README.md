# Arg sum 2

Реализуйте две функкции в файле [`arg-sum-2.S`](./arg-sum-2.S):

- `ArgSumShort(uint64_t, uint64_t*, uint64_t**)` – возвращает сумму трех своих аргументов по модулю `2^64`, где первый аргумент передан по значению, второй – по указателю, третий – по указателю на указатель.
- `ArgSumLong` – выполняет операцию аналогичную `ArgSumShort`, но работает с аргументами 6, 7 и 8, аргументы 1-5 игнорируются.
