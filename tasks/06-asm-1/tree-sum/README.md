# Tree sum

Рассмотрим бинарное дерево со следующей структурой узла:

```cpp
struct TreeNode {
    uint64_t value;
    TreeNode* left;
    TreeNode* right;
};
```

В ней указатели `left` и `right` хранят адреса левого и правого ребенка соответственно,
значение `nullptr` означает отсутствие соответствующего ребенка. Реализуйте функцию
`TreeSum`, которая по указателю на корень поддерева возвращает сумму значений `value`
у вершин этого поддерева по модулю `2^64`.

Аргумент функции `TreeSum` так же может быть равен `nullptr`, в таком случае функция должна
возвращать 0. Для доступа к полям структуры используйте `[rdi]`, `[rdi + 8]`
и `[rdi + 16]` соответственно (предполагая, что в `rdi` находится адрес структуры).
