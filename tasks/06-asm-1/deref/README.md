# Deref

Реализуйте функцию `uint64_t Deref(void* ptr, size_t count)`, которая интерпретирует
свой первый аргумент, как указатель типа

$$ \text{uint64\_t}\underbrace{* \ldots *}_{\text{count times}} $$

и возвращает значение типа `uint64_t`, на которое указывает этот указатель. Гарантируется, что
начальный и все промежуточные указатели не равны `nullptr`.

Не используйте рекурсию в этом задании.
